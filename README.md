# Multi-Model Voting System for Distinguishing AI-Generated vs Human-Created Painted Landscapes

This project implements a **multi-model voting system** to classify **AI-generated** and **human-created painted landscape images** using multiple machine learning and deep learning models. The final prediction is determined using a **soft-voting ensemble approach**.
